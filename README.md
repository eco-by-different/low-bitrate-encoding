# Low Bitrate Encoding Guide

This repository contains workflows and techniques focused on:

- Low bitrate video encoding (~100 MB per hour)
- AviSynth + x264 optimization
- Efficient media archival strategies
- System performance tuning (CPU / RAM)

---

## Philosophy

Goal:
- Minimal file size
- Maximum compatibility
- Acceptable visual and audio quality

Approach:
- Pre-processing (denoise, resize)
- Optimized encoding parameters
- Efficient audio compression (HE-AAC)

---

## Use Case

- Personal media archiving
- Low storage environments
- Playback on low-performance hardware
